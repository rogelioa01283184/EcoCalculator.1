
EcoCalculator

The main problem we are looking forward to addressing is that most people are not aware of how their daily consumption of resources like water, food, and energy impacts directly on our world and more specifically on climate change. Usually our activities lead to overconsumption, contributing to our planet’s degradation.

How to Run<br>
-Install Python
-Install the required library: pandas
-Run the program using the Python file
-The file results.csv will be created automatically in the same folder


Implemented Features -User input for daily habits (transport, food, electricity, waste)
-Score calculation using a point system
-Classification of impact level (Low, Medium, High)
-Personalized recommendations
-Data storage using a CSV file
-Use of the pandas library


Progress Explanation This program begins to address the problem of environmental awareness by allowing users to calculate their ecological footprint based on their daily habits. It provides feedback and stores the results, helping users better understand how their actions impact the environment. Additional Considerations The pandas library must be installed before running the program. The results.csv file is created automatically if it does not already exist.
